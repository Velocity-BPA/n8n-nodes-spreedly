# Streamline Payment Orchestration: Introducing the n8n-nodes-spreedly Community Node

We're excited to announce that Velocity BPA has released a new community node for n8n that integrates Spreedly's powerful payment orchestration platform directly into your automation workflows.

## The Payment Integration Challenge

Modern businesses often need to work with multiple payment gateways, manage sensitive payment data securely, and maintain flexibility in their payment infrastructure. Building and maintaining these integrations manually is time-consuming, complex, and risky. Teams face challenges with PCI compliance, vendor lock-in, and the technical overhead of managing multiple gateway APIs.

## Enter Spreedly + n8n

Spreedly is a payment orchestration platform that provides tokenization, multi-gateway routing, and centralized payment management. Our new n8n-nodes-spreedly community node brings these capabilities into your n8n workflows, enabling you to:

- **Tokenize payment methods securely** - Store and manage payment credentials without touching sensitive data
- **Route payments across multiple gateways** - Switch between payment providers without changing your integration
- **Orchestrate complex payment workflows** - Combine payment processing with your existing business automation
- **Maintain PCI compliance** - Reduce your compliance scope by leveraging Spreedly's vault

## Key Features

The n8n-nodes-spreedly node provides comprehensive access to Spreedly's API, allowing you to create payment methods, process transactions, manage gateways, and retrieve transaction data - all within your visual n8n workflows. Whether you're automating subscription billing, building custom checkout flows, or creating sophisticated payment routing logic, this node has you covered.

## Getting Started

Installing the Spreedly node is simple. In your n8n instance, run:


npm install n8n-nodes-spreedly


After installation, restart n8n and you'll find the Spreedly node available in your node palette. Connect it with your Spreedly credentials and you're ready to start building payment automation workflows.
